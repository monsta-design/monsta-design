用于用户基本的表单输入

## 模块引用

```ts
import { NSInputModule } from 'monsta-design/input';
```


## API


### [nsInput]

| 成员 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| `[nsInputSize]` | 输入框尺寸 | `default,sm,lg` | `default` | 
| `[nsInputPlaintext]` | 是否为纯文本显示 | `boolean` | `false` |
| `[nsInputColor]` | 是否为颜色选择器 | `boolean` | `false` |
